# 援助目录种子清单（待核实 — 不得直接录入 App）

_起草：2026-07-19（Claude 网络检索）。规则见 docs/decisions.md D-026：每条必须人工打电话核实 + 找到刊登号码的官方页面（sourceUrl）后，才能录入 `src/data/aidDirectory.json`。_

## 核实流程（每条都要做）

1. 打电话确认：号码能接通、确实提供所述服务、服务时间准确。
2. 找到**刊登该号码的官方网页**（政府/妇联/机构官网的具体页面，首页不算），作为 sourceUrl。
3. 录入 aidDirectory.json（格式照抄现有条目），`verifiedAt` 填核实当月。
4. 打不通或信息对不上 → 在下表标注后划掉，不录入。

## 重要背景（2026-07 检索发现，影响收录策略）

- **12338 已并入 12345**（2023-07 起全国话务并轨）：拨 12338 仍可转接，但各市单独的 12338 座机号正在淘汰。**不要**再按城市收录 12338 座机号，App 里保留全国 12338 一条即可。
- **12356 心理热线已覆盖全部 31 省**（2025-05 起）：城市级心理热线大多并轨，只收仍独立运营的专线（如北京 010-82951332）。
- 因此城市级条目的核心价值是：**实体法律援助中心**（有地址、直线电话、可当面办理）+ **本地公益机构/NGO** + 仍独立的专线。"区号+12348"这类号码价值低，重点核实直线电话和地址。

## 后续批次的官方数据源（按此模式逐省推进）

各省均有官方"法网"（`{省拼音缩写}.12348.gov.cn`，如 sc/js/hb/sh），站内有**法律援助机构查询**栏目，含全省各市中心的地址电话——这是最权威的批量来源。各市司法局官网通常也有同名栏目。

建议顺序（按目标用户密度）：重庆、天津、长沙、郑州、济南、沈阳、哈尔滨、昆明、南宁、贵阳……直至覆盖全部省会 + 计划单列市。

## 待检索的本地公益机构（NGO 类，价值最高也最难核实）

现有收录：北京源众（全国服务）、番禺妇联热线。待调研：各地反家暴/妇女法律援助公益机构（如律师公益项目、高校法律援助诊所）。**收录 NGO 前务必确认其仍在运营**——公益机构停运比政府机构频繁得多。
